# ETF投资组合回测系统

## 🔄 两种策略

### 策略A: 不再平衡（Buy and Hold）
- 定投结束后持仓不变，让市场自然发展
- 强势资产占比会逐渐提高

### 策略B: 定期再平衡
- 定投结束后每半年调整回目标仓位（25%/25%/25%/25%）
- 自动高抛低吸，控制集中度风险

## 📊 回测概述（策略A - 不再平衡）

### 投资策略
- **初始资金**: 100万元
- **时间范围**: 2015-01-05 至 2020-11-06 (5.84年)
- **投资方式**: 初始买入20% + 定投2年
- **定投周期**: 2015-01-05 至 2017-01-04 (474个交易日)
- **每日定投**: ¥1,687.76

## 💡 投资分析

### 优势
1. **分散投资**: 跨市场、跨地区配置降低单一市场风险
2. **定投策略**: 平滑买入成本，降低择时风险
3. **长期持有**: 年化收益9.45%，跑赢通胀
4. **美股表现突出**: 纳指涨幅190%，标普涨幅73%

### 风险提示
1. **疫情冲击**: 2020年3月最大回撤达26%
2. **A股表现较弱**: 红利指数和自由现金流指数涨幅相对较小
3. **汇率风险**: 美股投资受人民币汇率波动影响
4. **集中度偏移**: 纳指占比40%，高于目标25%

### 改进建议
1. 考虑定期再平衡，维持目标仓位
2. 增加债券类资产降低波动
3. 考虑分批止盈策略
4. 关注全球经济周期变化

## 🆚 策略对比

### 对比结果（2015-2020）

| 指标 | 不再平衡 | 定期再平衡 | 差异 |
|------|---------|-----------|------|
| **总收益率** | 69.14% | 63.58% | -5.56% |
| **年化收益率** | 9.45% | 8.83% | -0.63% |
| **最大回撤** | -26.03% | -27.56% | -1.53% |
| **夏普比率** | 0.086 | 0.066 | -0.020 |

### 最终资产配置

| 资产 | 不再平衡 | 定期再平衡 | 目标 |
|------|---------|-----------|------|
| 纳斯达克100 | 40.28% | 25.96% | 25% |
| 标普500 | 25.06% | 24.71% | 25% |
| 红利低波100 | 15.11% | 23.23% | 25% |
| 自由现金流 | 19.55% | 26.10% | 25% |

### 结论

✅ **在2015-2020牛市期间，不再平衡策略表现更优**
- 收益率高出5.56%
- 让强势资产（纳指）充分上涨
- 但纳指占比达40%，集中度风险较高

⚠️ **策略选择建议**
- **牛市/单边上涨**: 不再平衡策略，让强者更强
- **震荡市/不确定**: 定期再平衡，高抛低吸+风险控制
- **风险厌恶型**: 定期再平衡，维持组合平衡

## 🔧 使用方法
