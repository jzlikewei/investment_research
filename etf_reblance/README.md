# ETF投资组合回测系统

## 🔄 三种策略

### 策略A: 不再平衡（Buy and Hold）
- 定投结束后持仓不变，让市场自然发展
- 强势资产占比会逐渐提高

### 策略B: 定期再平衡（每半年）
- 定投结束后每半年调整回目标仓位（25%/25%/25%/25%）
- 自动高抛低吸，控制集中度风险
- 共7次再平衡

### 策略C: 阈值触发再平衡（推荐⭐）
- 定投结束后，当任一资产偏离目标超过5%时触发再平衡
- 触发条件：资产占比<20%或>30%
- 灵活应对市场变化，仅在必要时操作
- 共2次再平衡

## 💡 投资分析

### 优势
1. **分散投资**: 跨市场、跨地区配置降低单一市场风险
2. **定投策略**: 平滑买入成本，降低择时风险
3. **长期持有**: 年化收益9.45%，跑赢通胀
4. **美股表现突出**: 纳指涨幅190%，标普涨幅73%

### 风险提示
1. **疫情冲击**: 2020年3月最大回撤达26%
2. **A股表现较弱**: 红利指数和自由现金流指数涨幅相对较小
3. **汇率风险**: 美股投资受人民币汇率波动影响
4. **集中度偏移**: 纳指占比40%，高于目标25%

### 改进建议
1. 考虑定期再平衡，维持目标仓位
2. 增加债券类资产降低波动
3. 考虑分批止盈策略
4. 关注全球经济周期变化

### 策略分析

#### ✅ 推荐：策略C - 阈值触发再平衡

**核心优势：**
- 🎯 **夏普比率最高**（0.353），风险调整后收益最优
- 🛡️ **回撤最小**（-20.16%），风险控制最好
- 📉 **波动率最低**（17.51%），投资体验最平稳
- 🔧 **操作简单**，仅需2次再平衡

**再平衡记录：**
- 2018-06-11: 纳指达30.1%，触发再平衡
- 2020-02-03: 纳指30.7%、自由现金流19.7%，触发再平衡

**适合人群：**
- 希望获得稳健收益的投资者
- 注重风险控制的投资者
- 不想频繁操作的投资者

#### 策略A - 不再平衡

**适合场景：**
- 强烈看好某个市场（如美股）
- 愿意承受较高波动
- 追求最高收益

**风险提示：**
- 纳指占比从25%升至40%
- 集中度风险较高

#### 策略B - 定期再平衡

**适合场景：**
- 喜欢固定规律的操作
- 完全不想主动判断市场
- 偏好被动投资

**劣势：**
- 牛市中限制了强势资产涨幅
- 收益相对最低
